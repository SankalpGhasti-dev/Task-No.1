The repository includes comprehensive interview questions covering:

#### 2. Port Scanning Techniques
- TCP SYN scanning methodology
- TCP vs UDP scanning differences
- Understanding scan results and interpretations

#### 3. Security Measures
- Firewall configurations and port management
- Securing open ports and services
- Best practices for network security

#### 4. Network Analysis Tools
- Nmap usage and scanning techniques
- Wireshark for packet analysis
- Integration of scanning and analysis tools
